# 《最后一班地铁》

## 1. INT. 地铁站台 — 深夜

末班地铁的提示音在空旷的站台回响。荧光灯一盏盏熄灭，留下断续的光斑。

**林岸**，二十八岁，背着工业风格的相机背包，在长椅上低头查看手机，屏幕的冷光映在他的脸上。

**苏霓**，二十六岁，穿着深绿色长外套，拎着一个牛皮纸文件袋走过站台。她的脚步声在水泥地面上格外清晰。

她在距林岸三步远的地方停下。

**苏霓**
你是林岸吗？

**林岸**（抬头）
你怎么知道。

**苏霓**
你拍的那张照片救过我父亲的命。

林岸沉默地看着她，相机背包的拉链上挂着一枚生锈的金属牌——那是十年前一场矿难的纪念物。

**苏霓**（递出文件袋）
这是他留下的所有底片。他说，要交给愿意把它们公开的人。

地铁的远光灯从隧道深处刺出。

**林岸**
你为什么找我？

**苏霓**
因为还在拍这种事的人，已经不多了。

## 2. INT. 地铁车厢 — 连续

车厢里只有他们两个人。林岸打开文件袋，里面是几十张未冲洗的老式胶片，每张都标着日期和坐标。

林岸抽出最上面的一张对着光看。胶片上隐约是矿井入口的轮廓，井口边写着一串编号。

**林岸**
这些……不会让你父亲被报复吗？

**苏霓**
他已经走了。三年前。

车厢摇晃。她从口袋里摸出一颗薄荷糖，递给林岸。

**苏霓**
他喜欢吃这个。说在井下能压住那股铁锈味。

林岸接过糖，没有打开。

**苏霓**（压低声音）
我只问你一件事——发出去之后，你能保证自己也不出事吗？

**林岸**（看着窗外飞掠的隧道壁）
不能。但我会把它们寄到三家不同的报社，三个国家。

**苏霓**
那就够了。

广播响起：**「下一站，海港西路。本次列车终点站。」**

## 3. EXT. 海港西路出口 — 黎明

天还没亮。海风带着咸味从台阶外灌进来。

苏霓在出口前停下脚步，回头看林岸。她从口袋里摸出一支小型录音笔放在他掌心。

**苏霓**
最后一段录音在里面。如果哪天你听见枪声，就按下播放键。

她转身离开，长外套被风掀起。林岸站在台阶上，握紧文件袋和录音笔。

镜头拉远，海港的灯一盏接一盏熄灭，只剩第一缕晨光扫过站台的招牌——**「海港西路」**。

林岸抬起相机，对着空荡的台阶按下快门。

**FADE OUT.**
